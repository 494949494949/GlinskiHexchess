# Техническое задание
**Вариант 14.** Шестиугольные шахматы Глинского.
https://ru.qwe.wiki/wiki/Hexagonal_chess
## Требования
- Работоспособность
- Графический интерфейс
- Cохранение состояния (желательно в JSON)
# Реализация
Приложен проект **GlinskiHexchess**.
Код разбит по пакетам в зависимости от назначения:
- **logic**: ключевая логика игры, отделима от GUI.
- **gui**: графический интерфейс, всё связанное с рендерингом.
- **state**: функционал для сохранения состояния, в т.ч. через GUI
- **util**: вспомогательные утилиты
# Прочие заметки
- **Esc**: выход из игры
- **Ctrl+R**: перезапуск игры
- **Ctrl+S**: сохранение состояния
- **Ctrl+L**: загрузка состояния
